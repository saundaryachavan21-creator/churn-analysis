# 📊 Customer Churn Analysis and Customer Intelligence

A data analytics project that explores customer churn using **Python, SQL, Pandas, Matplotlib, and Seaborn**. The project focuses on data extraction, cleaning, feature engineering, exploratory data analysis (EDA), and visualization to uncover patterns that influence customer churn.

---

## 📌 Project Overview

Customer churn is one of the most important business metrics for subscription-based and customer-centric companies. This project analyzes customer data stored in a SQLite database to:

- Import and query customer data
- Clean and preprocess datasets
- Engineer meaningful features
- Perform exploratory data analysis (EDA)
- Create insightful visualizations
- Generate business insights related to customer churn

## 📊 Workflow

### 1. Data Import
- Connect to SQLite database
- Retrieve tables using SQL queries
- Load data into Pandas DataFrames

### 2. Data Cleaning
- Rename columns
- Remove unnecessary columns
- Convert data types
- Standardize categorical values
- Handle missing values

### 3. Feature Engineering
- Create new features
- Merge multiple datasets using SQL-style joins
- Export processed datasets

### 4. Exploratory Data Analysis (EDA)
- Analyze customer demographics
- Identify churn trends
- Explore customer behavior
- Generate summary statistics

### 5. Data Visualization
Visualizations created using:
- Matplotlib
- Seaborn

Examples include:
- Bar charts
- Count plots
- Distribution plots
- Correlation analysis
- Customer segmentation charts

### 6. Pivot Tables
- Summarize customer metrics
- Compare churn across different categories

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/customer-churn-analysis.git
```

2. Navigate to the project folder

```bash
cd customer-churn-analysis
```

3. Install dependencies

```bash
pip install pandas numpy matplotlib seaborn
```

4. Launch Jupyter Notebook

```bash
jupyter notebook
```

5. Open:

```
churn_analysis.ipynb
```
